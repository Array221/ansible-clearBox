ClearBox
========

Ansible role that automates basic configuration of Debian servers.

Requirements
------------

This role was tested on Debian 11, but it should work on any Debian-based distro.

Role Variables
--------------

### Required

None (yet)

### Optional

None (yet)

Dependencies
------------

This role does not require other dependencies.

Example Playbook
----------------

```yaml
---
  hosts: all
  vars: []
  roles:
    - clearBox
```
